# Lista 00 – Beecrowd (Linguagem C)

Esta pasta contém a **Lista 00 de exercícios do Beecrowd**, desenvolvida na disciplina de **Programação de Computadores** do curso de **Ciência da Computação – Unisantos**.

Os exercícios são **introdutórios na linguagem C** e têm como objetivo apresentar os principais conceitos básicos de programação.

---

## Objetivos da lista

Durante a resolução dos exercícios são praticados conceitos fundamentais como:

- uso de **bibliotecas da linguagem C** (ex: `<stdio.h>`)
- **entrada e saída de dados** com `scanf` e `printf`
- **declaração de variáveis**
- **operações matemáticas básicas**
- **formatação de saída**
- **lógica básica de programação**

Esses exercícios ajudam a construir a base necessária para problemas mais complexos no futuro.
