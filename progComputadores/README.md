# 💻 Programação de Computadores - UniSantos

Bem-vindo ao diretório principal da disciplina de **Programação de Computadores**! Este espaço centraliza os códigos, anotações e resoluções de problemas da matéria.

---

## 📌 Sobre a Matéria
Esta disciplina é o ponto de partida do 1º Semestre (2026/1) para as turmas de **Ciência da Computação** e **Sistemas de Informação** da Universidade Católica de Santos. 

**Ementa e Objetivos:**
* A matéria introduz os conceitos fundamentais de linguagens de programação e estuda o funcionamento de uma linguagem de alto nível.
* Explora comandos básicos, tipos de dados (escalares e estruturados), operadores matemáticos, controle de fluxo e subprogramas.
* Desenvolve o conceito prático de organização de arquivos e armazenamento de dados em memória secundária.

**Competências Desenvolvidas:**
* **Identificar Problemas:** Alterar a perspectiva de "usuário" para "desenvolvedor", enxergando problemas passíveis de solução algorítmica considerando os limites da computação.
* **Implementar Sistemas Computacionais:** Usar boas práticas e engenharia de software para atender regras de negócio usando recursos comuns da maioria das linguagens de alto nível.
* **Resolver Problemas:** Habilidade de resolver cenários reais no limite do contexto da Computação como máquina.

## 🛠️ Ambiente de Desenvolvimento
O desenvolvimento das atividades e exercícios desta disciplina foi adaptado de acordo com o ambiente de estudo:

* 🏠 **Em casa:** Utilizei o **Visual Studio Code (VS Code)**. Para compilar e executar os códigos em C diretamente pelo terminal integrado, o padrão utilizado foi:
  * **Compilação:** `gcc nome_do_arquivo.c -o nome_do_arquivo`
  * **Execução:** `./nome_do_arquivo`
* 🏫 **Na faculdade:** Durante as aulas práticas nos laboratórios, a IDE utilizada foi o **Code::Blocks**.
* 🐝 **Plataforma de Submissão:** Vários problemas e desafios algorítmicos foram testados e validados no **Beecrowd** (antigo URI Online Judge). 
  * 🔗 Você pode acompanhar meu progresso e resoluções aqui: [Meu Perfil no Beecrowd](https://judge.beecrowd.com/pt/profile/1217522).

Para compilar e executar os códigos pelo terminal integrado, utilizamos:
* **Compilação:** `gcc nome_do_arquivo.c -o nome_do_arquivo` (ou similar)
* **Execução:** `./nome_do_arquivo`
