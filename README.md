# 🎓 Ciência da Computação — UNISANTOS

Este repositório reúne códigos, exercícios e anotações produzidos durante minha graduação em **Ciência da Computação na Universidade Católica de Santos (UNISANTOS)**.

A ideia é organizar e documentar o aprendizado ao longo do curso, reunindo principalmente conteúdos **práticos**, como:

* exercícios de programação
* implementações de algoritmos
* experimentos com linguagens de programação
* projetos acadêmicos
* resoluções de problemas
* anotações técnicas em Markdown

Este projeto funciona como um **registro da evolução ao longo da graduação**.

---

# 📚 Estrutura do Repositório

Cada disciplina relevante do curso terá seu próprio diretório dentro deste repositório.

Cada pasta pode conter:

* códigos fonte
* listas de exercícios
* diagramas
* anotações
* um `README.md` explicando a disciplina e os conteúdos estudados

Exemplo de organização:

```
Ciencia-da-computacao-unisantos/

progComputadores/
algoritmos/
banco-de-dados/
estrutura-de-dados/
sistemas-operacionais/
redes/
```

Cada disciplina terá sua própria organização interna dependendo do tipo de atividade desenvolvida.

---

# 💻 Disciplinas com foco prático

O objetivo deste repositório é concentrar principalmente disciplinas que envolvem **implementação, programação ou resolução prática de problemas**, como por exemplo:

* Programação de Computadores
* Algoritmos
* Estruturas de Dados
* Banco de Dados
* Sistemas Operacionais
* Redes de Computadores
* Engenharia de Software

Matérias mais teóricas nem sempre terão conteúdo aqui, a menos que envolvam algum tipo de implementação ou material útil para estudo.

# 📂 Disciplinas atuais

## 💻 Programação de Computadores

Diretório:

```
progComputadores/
```

Esta disciplina introduz os fundamentos da programação utilizando **linguagem C**, incluindo:

* variáveis
* operadores
* estruturas de controle
* vetores e matrizes
* manipulação de dados
* arquivos

O diretório contém exercícios práticos desenvolvidos ao longo das aulas.

Cada exercício está implementado em arquivos `.c`.
